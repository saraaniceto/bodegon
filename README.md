Durante a Imersão Dev Alura com Google Gemini, os participantes tiveram como objetivo desenvolver um website com a função de base de conhecimento, com tema de livre escolha. A Bodegón - Galeria de Natureza Morta, surge como parte de pesquisa pessoal reunindo obras de curadoria previamente realizada com nova base de dados alimentada com auxílio da ferramenta de IA.

Com o objetivo de promover os primeiros contatos com programação web, utilizando fundamentos de HTML, CSS e JavaScript, a Imersão ofereceu um projeto base detalhadamente explicado durante as aulas, incluindo o projeto de integração Node.JS e API do Google (os quais não serão apresentados nesse repositório, ver figuras abaixo)

Dentro das tarefas e requisitos para submissão do projeto final da Imersão, para esse projeto foram desenvolvidos individualmente as seguintes alterações:
- Alterações no HTML e CSS de forma a aplicar identidade visual do projeto utilizando auxílio do Gemini Code Assist
- Criação de prompt para integração com API Google AI Studio (ver figuras abaixo)
- Criação de função que embaralha o JSON antes de exibir na página, utilizando auxílio do Gemini Code Assist
